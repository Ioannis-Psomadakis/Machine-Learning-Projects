# Support Vector Machine Project

In this project, the breast cancer dataset that is provided in the sklearn.datasets library will be used.

Firstly, the method of 'Support Vector Machine' will be implemented and then the effects of the application of Normalization on the results of the method will be examined.


```python
# Import of the needed libraries
from sklearn.model_selection import train_test_split
from sklearn.svm import SVC  # 'Support Vector Classifier'
from sklearn.datasets import load_breast_cancer  # Import of the breast cancer dataset
```


```python
# Object creation
cancer = load_breast_cancer()
```


```python
# Dataset splitting
X_train, X_test, y_train, y_test = train_test_split(cancer.data, cancer.target, random_state = 0)
```


```python
# SVM method import
svc = SVC()
```


```python
# Model feed
svc.fit(X_train, y_train)
```

Now, the effects of the Normalization will be examined


```python
# Normalization steps
```


```python
# Calculation of minimum value per attribute of the training set
min_on_training = X_train.min(axis = 0)
```


```python
#  Calculation of the range of each attribute (max - min) in the training set
range_on_training = (X_train - min_on_training).max(axis = 0)
```


```python
# Subtracting the min and dividing by range
# Then there will be min = 0 and max = 1 for each attribute
X_train_scaled = (X_train - min_on_training) / range_on_training
X_test_scaled = (X_test - min_on_training) / range_on_training
```


```python
# Refeeding the model
svc = SVC()
svc.fit(X_train_scaled, y_train)
```
